# Protein Design Tools - Skill Files

This repository contains skill files (markdown format) for various protein design and analysis tools. Each skill file provides comprehensive documentation on how to use the tool via Linux command line, including installation, basic usage, input/output organization, and output interpretation.

## Available Skill Files

### Core Protein Design Tools
1. **ProteinMPNN** (`skills/proteinmpnn.md`) - Neural network for protein sequence design
2. **RFDiffusion** (`skills/rfdiffusion.md`) - Diffusion model for protein structure generation
3. **ESM2** (`skills/esm2.md`) - Protein language model for embeddings and predictions
4. **AutoDock Vina** (`skills/autodock_vina.md`) - Molecular docking for drug discovery
5. **OpenMM** (`skills/openmm.md`) - Molecular dynamics simulation toolkit
6. **TM-align** (`skills/tmalign.md`) - Protein structure alignment and comparison

### Additional Tools (to be created)
- Chai1

## Skill File Structure

Each skill file follows this structure:

1. **Overview** - Brief description of the tool
2. **Installation** - How to install the tool
3. **Basic Usage** - Command-line interface examples
4. **Input Requirements** - Input file formats and preparation
5. **Output Structure** - Generated files and their organization
6. **Output Interpretation** - How to analyze and interpret results
7. **Advanced Usage** - Advanced features and workflows
8. **Troubleshooting** - Common issues and solutions
9. **Integration** - How to combine with other tools
10. **References** - Links to documentation and papers

## Purpose

These skill files are designed to:
- Provide LLMs with detailed knowledge of protein design tools
- Enable LLMs to generate correct command-line invocations
- Help LLMs understand input/output requirements
- Allow LLMs to interpret analysis results
- Facilitate automated workflow generation

## Usage Example

An LLM can use these skill files to generate complete workflows:

```bash
# Example workflow generated by LLM using skill files
# 1. Generate protein structures with RFDiffusion
python scripts/run_inference.py --contigs "100" --out ./designs --num_designs 10

# 2. Design sequences with ProteinMPNN
for pdb in designs/*.pdb; do
    python protein_mpnn_run.py --pdb_path "$pdb" --out_folder ./sequences
done

# 3. Score designs with ESM2
python scripts/extract.py esm2_t33_650M_UR50D sequences.fasta ./embeddings

# 4. Compare to reference with TM-align
TM-align reference.pdb best_design.pdb -o alignment.txt
```
